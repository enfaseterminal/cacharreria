¿Cómo usar?

  1. Ingresa el nombre de tu red WiFi (SSID)
  2. Ingresa tu contraseña WiFi (si la tienes)
  3. Selecciona el tipo de seguridad
  4. Haz clic en "Generar Código QR"
  5. Descarga o comparte el código QR
